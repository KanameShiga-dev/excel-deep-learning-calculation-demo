# 第4回 勾配降下法

## この回で学ぶこと

学習率で更新ステップとLoss推移が変わることを見ます。パラメータが少しずつ動く様子を表で確認します。


## 含まれるファイル

- `excel_gradient_descent_demo.xlsx`
- `excel_gradient_descent_demo_code.py`
- `excel_gradient_descent_slides.pptx`
- `excel_gradient_descent_operation_explainer.pptx` - デモ操作解説スライド
- `README.md`

## 使い方

1. `excel_gradient_descent_demo.xlsx` を開く。
2. `01_INPUT` の入力表と設定表を確認する。
3. `excel_gradient_descent_demo_code.py` を外部ファイルとして開き、内容をPython in Excelへ貼り付けて実行する。
4. 返ってきた `result_df` を確認する。
5. Excel上の値や `display_mode` を変えて、返却表がどう変わるかを見る。


## Excelで変更できるもの

| 場所 | 変更・確認するもの | 見える変化 |
|---|---|---|
| `01_INPUT A6:B8` | `data_df` | 入力と正解 |
| `01_INPUT D6:F9` | `settings_df` | 学習設定 |

## 注意

この教材は、計算の流れを小さな表で確認するための補助教材です。
